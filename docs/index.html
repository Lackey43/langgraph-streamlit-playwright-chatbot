What makes this project stand out

LangGraph Agentic Flow

Custom StateGraph with intelligent routing, tool calling, file context injection, and structured memory management. Not a simple chain.

Playwright Web Tools

Real browser automation for web search, page browsing, and content extraction. Handles modern JavaScript sites reliably.

Multi-Modal File Support

Upload PDFs (text + tables), images (OCR + vision), DOCX, CSV, TXT. Smart summarization injected into context.

SMART MEMORY

Only keeps the last 6 states

Per-user SQLite database that automatically trims conversation history to the most recent 6 turns. This keeps context windows efficient and storage costs low while maintaining conversational continuity.

Automatic trimming
Old turns are deleted on every new message
Multi-user isolation
Each user has their own conversation history
# Memory trimming logic
DELETE FROM turns 
WHERE user_id = ? 
  AND turn_id < (
    SELECT MIN(turn_id) FROM (
      SELECT turn_id FROM turns 
      WHERE user_id = ? 
      ORDER BY turn_id DESC 
      LIMIT 6
    )
  )
